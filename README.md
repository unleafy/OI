# OI Code Repository

本仓库用于日常 OI / 算法训练代码管理与记录。

---

## 📁 目录结构

.
├── main.cpp      # 日常练习代码文件
├── stored/       # 已通过题目代码存档
└── record.md     # 做题记录

---

## 🧠 代码规则

### main.cpp

- 日常所有代码均写在 main.cpp
- 用于：
  - 思路验证
  - 调试算法
  - 赛时快速实现

---

### stored/

通过的题目代码统一归档，命名规则：

{platform}-{problem-id}.{ext}

示例：

luogu-P1001.cpp
codeforces-1872A.cpp
atcoder-abc301C.cpp

说明：

- platform：luogu / codeforces / atcoder 等
- problem-id：题号或编号
- ext：一般为 cpp（或 py 等）

---

### record.md

记录每日训练情况，包括：

- 题目
- 完成情况
- 简要思路 / 错误总结
